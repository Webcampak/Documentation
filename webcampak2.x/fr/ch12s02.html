2. Installation de Webcampak

Une fois Ubuntu Server installé, téléchargez la dernière version de Webcampak et décompressez le répertoire.

Exécutez ensuite le script d'installation qui se trouve dans le répertoire webcampak2/init/install/

wpak@webcampak:~$ bash webcampak2/init/install/install.Ubuntu12.04.sh

Suivez ensuite l'ensemble des instructions qui seront affichées à l'écran.

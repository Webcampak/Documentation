2. Sélection de la source

En haut à gauche de la fenêtre vous pouvez sélectionner la source que vous souhaitez afficher, seules les sources pour lesquelles vous disposez de droits d'accès seront affichées ici.

Si une source est manquante demandez à votre administrateur de vérifier les permissions de votre compte utilisateur.

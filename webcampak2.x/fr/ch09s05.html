5. Gestion des groupes

Webcampak permet d'ajuster de manière très fine les groupes et permissions de ses utilisateurs.

En cliquant sur le nom d'un groupe une nouvelle interface vous permet de configurer les membres de groupe ainsi que les permissions associées.

Ce menu vous permet d'ajouter par glisser déposer des utilisateurs au sein du groupe et de définir quelles sont les pages et fonctionnalités qui seront autorisées pour les membres du groupe.

Liste des pages disponible au sein du Webcampak:

  • auth.base.sources: indispensable, permet à l'utilisateur d'obtenir une liste des sources qu'il a le droit de consulter

  • auth.base.lang: indispensable, permet à l'utilisateur d'obtenir l'interface dans la langue de son choix

  • auth.stats: affiche le menu "stats" permettant de consulter les statistiques

  • auth.stats.system: affiche les statistiques de l'ensemble système

  • auth.stats.sources.disk: affiche les statistiques d'utilisation disque pour la source sélectionnée

  • auth.stats.pictures.files: affiche le nombre de clichés capturés pour la source sélectionnée

  • auth.stats.pictures.size: affiche la taille des clichés capturés pour la source sélectionnée

  • auth.view.pictures: affiche les images associées à la source

  • auth.view.pictures.email: permet d'envoyer un email contenant la photo actuellement visionnée

  • auth.view.pictures.comment: permet d'attacher un commentaire à l'image

  • auth.view.videos: affiche les vidéos associées à la source

  • auth.config: affiche le menu permettant d'accédder à la configuration

  • auth.config.logs: affiche les logs correspondant à la source sélectionnée

  • auth.config.instantcapture: permet à l'utilisateur de déclencher une capture instantannée sur la source sélectionnée

  • auth.config.sources: permet d'accédder au menu de configuration des sources

  • auth.config.sources.read: permet de consulter les paramètres de configuration de la source sélectionnée

  • auth.config.sources.write: permet de modifier les paramètres de la source sélectionnée

  • auth.config.admin.general: permet d'accédder au menu de configuration système

  • auth.config.admin.permissions: permet de modifier les permissions du Webcampak

  • auth.config.admin.reboot: permet de redémarrer le Webcampak

  • auth.config.admin.connecteddevices: permet de consulter la liste des périphériques physiquement raccordés au Webcampak.

Veuillez noter que la fonction d'ajout de pages est à usage interne et il est très peu probable qu'elle vous soit utile.

Note

Webcampak est fourni pré-installé et pré-configuré, ce chapitre, à destination de nos testeurs, est fourni à titre d'information et ne vous concernera probablement pas.

Webcampak est composé d'un coeur développé en Python et d'un ensemble de logiciels tiers permettant de réaliser des manipulations complémentaires.

Le coeur peut être considéré comme le chef d'orchestre coordonnant l'ensemble des actions réalisées par le Webcampak, néanmoins les logiciels tiers doivent êtres installés et configurés correctement pour permettre la bonne marche de l'ensemble.

Webcampak est conçut pour fonctionner avec le système d'exploitation Ubuntu Server, et est fourni avec un ensemble de scripts permettant aux logiciels complémentaires de fonctionner correctement avec Webcampak. Il est très probable que Webcampak fonctionne avec un système d'exploitation autre que Ubuntu Server néanmoins soyez prudent vis-à-vis de la configuration des logiciels complémentaires, il reste possible que nos fichiers de configuration ne soient pas compatible avec d'autres versions.

De plus nous vous conseillons très fortement d'installer Webcampak sur un système d'exploitation fraîchement installé.

1. Installation d'Ubuntu Server

Téléchargez et installez Ubuntu Server, veillez à ne pas ajouter de paquets complémentaires mis à part OpenSSH serveur qui vous permettra de prendre contrôle à distance.

Nous avons testé la compatibilité avec les versions 12.04

Pour ce chapitre, nous utiliserons le nom d'utilisateur système "wpak".

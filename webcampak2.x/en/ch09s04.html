4. Manage Sources

Source management is slightly more complex.

You can use this tab to add new sources within Webcampak, those sources will be automatically pre-configured and be ready to use by Webcampak.

To add a source, simply click on "Add"

You will need to complete three different elements, those are very important and used by a large number of other Webcampak sections:

  • Source ID: is the internal ID used by Webcampak, this number must be unique. No other source should have the same number. Please note that an ID cannot be changed afterwards.

  • Source Name: this is the name you can use to identify the source within Webcampak interface, we try to use the name wherever is possible.

  • Weight: is the weight of the source compared to other sources, source with the lowest weight are displayed first. If two source have the same weight, they will be sorted by alphabetical order.

Please note that for any created source, "root" user is automatically added as a source member.

If you click on a source, you will see a menu that will let you drag users into the source.

By default, users attached to a source get "Read/Write" permissions to content of this source.

If you double click on a source you can edit its name or weight.
